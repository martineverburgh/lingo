Info Lingo

Lingo is een spel waarbij de speler een woord van 5 letters invoert. De computer controleert vervolgens welke letters van het opgegeven woord overeenkomeen met de letteer van het te raden woord. Met rood worden de letters die op de juiste plaats staan, met geel worden juiste letters aangegeven die niet op de goede plaats staan.
De speler krijgt 5 pogingen om het juiste woord te raden.

Info Memory

Bij Memory is het de bedoeling om steeds twee dezelfde plaatjes te vinden door op twee geloten vakjes te klikken. Wanneer de speler twee dezelfde vakjes aanklikt, blijven deze open staan.
Het is de bedoeling om in zo weinig mogelijk pogingen alle vakjes om te draaien.
